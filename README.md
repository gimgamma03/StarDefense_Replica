# StarDefense_Replica

게임 스타디의 리소스를 활용한 과제 프로젝트
---

## 🎮 프로젝트 개요
- Unity 기반 2D 타워 디펜스 게임
- 웨이브 단위로 적이 등장하고, 타워를 설치/조합하여 방어
- 자원(골드, 미네랄)을 관리하며 일꾼을 추가로 소환 가능
- 게임오버 및 UI 시스템 포함
- 현재 과제용 기능위주의 제작으로 미네랄 소모처 등의 밸런스는 안돼있는 상태
---

## 📂 주요 클래스 역할

### `GameManager`
- 게임 전체 상태 관리 (골드, 라이프, UI)
- 타워 선택 및 조합 버튼 처리
- 게임오버 UI 활성화
- 일꾼(Miner) 소환 기능 제공

### `SpawnManager`
- 웨이브 진행 및 적 소환 관리
- 일반 적 및 현상금(Bounty) 적 소환
- 살아있는 적 리스트(`ActiveEnemies`) 관리
- 웨이브 종료 조건 체크

### `EnemyUnit`
- 적 기본 유닛 클래스
- 체력, 이동 속도, 스프라이트 등 초기화
- 웨이포인트를 따라 이동
- 데미지 처리 및 사망 처리 (`Die`)
- 사망 시 골드 지급 및 리스트에서 제거

### `BountyEnemyUnit`
- 현상금 유닛

### `Commander`
- 사령관 유닛
- 일정 간격으로 가장 가까운 적을 탐색 후 공격
- 적이 목표 지점에 도달했을 때 라이프 감소 처리

### `TowerManager`
- 타워 설치/제거/조합 관리
- 좌표별 타워 딕셔너리 관리
- 등급별 타워 리스트 관리
- 타워 조합 시 상위 등급 타워 생성
